# emifla/app

This repository was initialized by GitHub Copilot to add a SwiftUI starter app for iPhone.

What I added in branch `feature/add-swiftui-starter`:
- Models.swift — data models for To-Do and Budget features
- DataController.swift — persistence and update logic for daily budgets and weekly resets
- Views.swift — SwiftUI views and app entrypoint

Instructions: create a new iOS App in Xcode (SwiftUI lifecycle). The files in the branch implement a working starter app with JSON persistence.
